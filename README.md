# Neemsi Restaurant 🍽️

A restaurant management and billing system built using C programming. This project is designed to manage menu items, handle customer orders, and generate bills efficiently through a command-line interface.

## Features

* Display restaurant menu
* Add customer orders
* Calculate total bill automatically
* Generate itemized bill
* Manage multiple menu items
* Simple and efficient command-line interaction

## Tech Stack

* C

## Purpose

This project was created to strengthen programming fundamentals and apply concepts like arrays, functions, loops, and conditional statements in a real-world management system.

## How It Works

* View the available menu items
* Select food items to order
* Enter the quantity of each item
* The system calculates the total price
* Generates the final bill for the customer
