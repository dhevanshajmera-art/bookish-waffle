# Neemsi Restaurant Management System

A console-based Restaurant Management System written in C.
This project allows customers to order food from different menus, calculate bills, apply discounts, and generate restaurant invoices.

## Features

- AC and Non-AC section selection
- Breakfast Menu
- Lunch & Dinner Menu
- Around-The-Clock Menu
- Multiple food categories
- Automatic bill generation
- GST calculation
- Lucky number discount game
- Order logging system
- Table number and bill number management

## Technologies Used

- C Programming Language
- Standard C Libraries
  - stdio.h
  - stdlib.h
  - time.h

## How to Run

### Compile the Program

```bash
gcc abc.c -o restaurant
```

### Run the Program

```bash
./restaurant
```

For Windows:

```bash
restaurant.exe
```
